To enable hands free functioning of mobile, most of the work can be done by just speech. 

Installation and Usage:
1)	Install the app on android enabled device.
2)	Device should have mic
3)	Launch the App
4)	Press the Start Speech Button and say the “command”, app will do accordingly

Here command includes:
	a)	Any app name : this will open that app if pronounced correctly
	b)	Wifi on : turn on the wifi
	c)	Wifi off: turn off the wifi
	d)	GPS: open the GPS setting page where you can turn on the GPS, it requires manual intervention as required                  by android security feature
	e)	Lock: will lock the screen --> will require admin permissions, will be asked while intallation
	f)	Call ‘name’ :  will start call to the name in contact if matches exactly
